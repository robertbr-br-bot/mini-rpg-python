# Mini RPG Python
Eu desenvolvi um mini RPG em Python com sistema de combate, eventos aleatórios e lógica de jogo em modo texto.

Eu desenvolvi este projeto como forma de praticar lógica de programação utilizando Python, simulando um jogo simples em modo texto com sistema de combate e exploração.

## Sobre o projeto

Este é um jogo interativo em terminal onde o jogador pode explorar caminhos, enfrentar inimigos e tomar decisões durante a partida. O foco do projeto foi aplicar conceitos fundamentais de programação de forma prática.

## Funcionalidades

- Sistema de combate com cálculo de dano
- Chance de dano crítico (tanto para o jogador quanto para o inimigo)
- Uso de poções para recuperação de vida
- Sistema de exploração com eventos aleatórios
- Gerenciamento de atributos (vida, ouro e inventário)
- Interação com o usuário via terminal

## Tecnologias utilizadas

- Python
- Biblioteca `random`

## Objetivo do projeto

Meu objetivo com esse projeto foi praticar e consolidar conceitos importantes como:

- Estruturas condicionais (`if`, `elif`, `else`)
- Laços de repetição (`while`)
- Entrada de dados com `input`
- Controle de fluxo do programa
- Organização da lógica de um sistema interativo
